### [bouncing.loader.css](bouncing.loader.css)

#### Usage
```html
<link rel="stylesheet" type="text/css" href="bouncing.loader.css">
<div class="bouncing-loader">
    <div></div>
    <div></div>
    <div></div>
</div>
```
